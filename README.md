# Raspberry-Pi-Thermostat-System

## Project Summary
This project is a Raspberry Pi thermostat system that reads temperature data from a sensor and controls heating and cooling states. It uses an LCD display to show time, temperature, and system state, and LED indicators to represent heating and cooling activity. The system also sends serial updates to a server simulator.

## What I Did 
I successfully integrated the temperature sensor, LCD display, and button controls into one working system. The state machine logic functions correctly and the LEDs respond based on temperature and set point. Serial communication to the simulator also works consistently.

## Areas for Improvement
I could improve code organization and reduce debugging time by testing smaller sections earlier. Better planning before implementing larger features would help avoid minor errors like indentation issues.

## Tools and Resources Used
Python, Raspberry Pi GPIO libraries, I2C communication, serial communication, and GitHub for version control.

## Transferable Skills
This project strengthened my debugging skills, hardware-software integration, state machine implementation, and structured problem-solving.

## Maintainability
The project is structured using classes and separate methods to keep logic organized and readable. Clear naming and modular design make the system easier to modify or expand in the future.
